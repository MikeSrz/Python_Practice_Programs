DOCUMENTACIÓN RÁPIDA PARA EL USO DE LA BIBLIOTECA MY_TOOLS

Recordemos la etructura para importar funciones:
	- from mytools.fichero.py import function:
Puedes ponerle tus propios alias así:
	- from mytools.fichero.py as fich import function as func

Consta de ficheros de un tipo, dentro se hayan funciones relacionadas con el tema:

	- algorith.py //Funciones donde se manejan soluciones con Algoritmos reconocidos.
	- maths.py //Funciones para hacer comprobaciones o cálculos matemáticas.
	- dates.py //Funciones para el manejo de fechas.
	- graphics.py //Funciones para crear gráficos, dibujos...
	- file_handling.py //Funciones para manejar ficheros
	- statistics.py //Funciones de estadística.
